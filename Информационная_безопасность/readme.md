**Проекты по ИБ**

***************************************************************************************************************************************************

**Аналитика антивирусной защиты**

Источник анализа - логи Kaspersky Security Center.

1. Активность угроз ИБ по часам (пиковая нагрузка).
   
2. Наиболее часто блокируемые домены (детализация).

3. Аномальные события (детализация).
   
4. Ошибки SSL-сертификатов (детализация).

5. Заблокированные угрозы (детализация).

***************************************************************************************************************************************************

**Аналитика безопасности веб-доступа**

Источник анализа - логи UserGate.

1. Поиск подозрительных пользователей.

2. Поиск потенциально вредоносных файлов.

3. Поиск подозрительного исходящего трафика.
 
4. Поиск нарушений политики безопасности.

5. Аномальная активность в ночное время.

6. Активные алерты безопасности.

7. Исходящий трафик по часам.

***************************************************************************************************************************************************

**Анализ жизненного цикла сертификатов КриптоПро**

Источник анализа - логи КриптоПро.

1. Определение статусов сертификатов.

2. Определение организаций.

3. План замены сертификатов на ближайшие 3 месяца.

4. Перечень истекших сертификатов.
