⭕**Среднее время на решение 1 задачи (файл codesubmit.csv)**

Поиск среднего времени, которое тратится пользователями на полное, успешное решение одной задачи. Есть некоторые пользователи, которые способны с первого раза решить задачу - таких пользователей считать не нужно, будем считать их гениальными, но "выбросами".

**_Пример выполнения:_**

611.86

⭕**Среднее время нахождения на платформе (файл codesubmit.csv)**

Поиск среднего времени в часах, которое проводит пользователь в день на платформе.

**_Пример выполнения:_**

1.7

⭕**Среднее количество задач за один активный сеанс (файл codesubmit.csv)**

Поиск среднего количества задач, которые решается за один активный сеанс. Активный сеанс - период, когда между любой активностью пользователя разница менее или равна часу, не более.

**_Пример выполнения:_**

3.14

⭕**Самый популярный час на платформе (файл codesubmit.csv)**

Поиск самого "популярного" часа дня на платформе (активности пользователей).

**_Пример выполнения:_**

16
