⭕**Python-Итоговый_проект (загрузка с API, обработка, агрегирование и выгрузка в GoogleSheets, выгрузка данных в БД Postgres, email-оповещение, логирование)**

Скрипт реализует (api_2026-01-13_11_28.log, output_.txt, Stats.xlsx):

1.обращение к API для получения данных;

2.обработку и подготовку данных к загрузке в БД;

3.загрузку обработанных данных в локальную БД PostgreSQL (разворачивала в Colab) (имеется проверка на уникальность записей, дублирующиеся записи перезаписиваются, date-поле "loaded_at" обновляется);

4.сохранение таблицы из БД в текстовый файл output.txt для удобства просмотра;

5.сохранение лога работы скрипта во время обработки данных с отлавливанием всех ошибок, выводом промежуточных стадий, трекингом времени; сохранение лога в текстовый файл, его именование в соответствии с текущей датой - api_2026-01-13_11_28.log (в папке с логами остаются только логи за последние 3 дня);

6.агрегирование данных за день: количество совершенных попыток, количество успешных попыток, количество уникальных юзеров, среднее количество успешных попыток на юзера;

7.выгрузку агрегированных данных в Google Sheets;

8.оповещение работы скрипта на email (mail.ru) со ссылкой на таблицу Google Sheets.

**_Пример выполнения:_**

Всего попыток за день {'2023-04-01': 134}

Успешных попыток за день {'2023-04-01': 23}

Количество уникальных юзеров за день {'2023-04-01': 30}

Среднее количество решенных задач юзера за день {'2023-04-01': 0.77}
