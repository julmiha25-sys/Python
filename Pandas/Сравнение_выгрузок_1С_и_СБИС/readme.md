**Сравнение выгрузок СБИС и 1С**

1. Выгрузка из СБИС находится в папке Входящие, они загружаются в формате csv в один датафрейм (ДС). 

   1.1. Названия столбцов из нескольких слов сцеплены через нижнее подчеркивание в одно длинное слово.

2. Файлы с выгрузкой из аптек находятся в папке Аптеки/csv/correct/. Каждый файл формата csv загружается вручную на обработку отдельно в датафрейм аптеки (ДА). 

   2.1. В ДА добавлены столбцы:"Номер счет-фактуры", "Сумма счет-фактуры", "Дата счет-фактуры" и "Сравнение дат".

   2.2. В каждой строке ДА:

        2.2.1.Если "Поставщик" - ЕАПТЕКА, то к "Номер накладной" добавлено "/15".
   
        2.2.3.Поиск записей из ДС по данному номеру накладной: 

        2.2.3.1. Из найденных строк оставляются только те, которые имеют один из типов документа: ["СчФктр", "УпдДоп", "УпдСчфДоп", "ЭДОНакл"].

        2.2.3.2. Если ничего не найдено - осуществляется переход к следующей строке. Если найдено - сохраняются значения Номер, Сумма и Дата.

        2.2.3.3. Даты переводятся в формат dd.mm.yyyy (для унификации).

        2.2.3.4. Найденные для данной строки значения из столбцов пункта 2.2.3.2 записываются в столбцы из пункта 2.1.

        2.2.3.5. В столбец "Сравнение дат" помещается "Не совпадает!", если найденная дата и дата накладной отличаются, в противном случае - пустая строка.
        Если по заданным условиям находится несколько подходящих документов - выбирается значение из первого найденного. Это означает, что если у документов разные суммы, берется та, что стоит первой в выгрузке из СБИС.
