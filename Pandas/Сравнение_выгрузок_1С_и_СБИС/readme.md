**Сравнение выгрузок СБИС и 1С**

Выгрузка из СБИС находится в папке Входящие, они загружаются в формате csv в один датафрейм (ДС). 

Названия столбцов из нескольких слов были сцеплены через нижнее подчеркивание в одно длинное слово.

Файлы с выгрузкой из аптек находятся в папке Аптеки/csv/correct/. Каждый файл формата csv загружается на обработку отдельно в датафрейм аптеки (ДА). 

В ДА добавлены столбцы:"Номер счет-фактуры", "Сумма счет-фактуры", "Дата счет-фактуры" и "Сравнение дат".

В каждой строке ДА:

1.Если "Поставщик" - ЕАПТЕКА, то к "Номер накладной" добавлено "/15".

2.Найдены все записи из ДС по данному номеру накладной: 

2.1. Из найденных строк нужно оставлены только те, которые имеют один из типов документа: ["СчФктр", "УпдДоп", "УпдСчфДоп", "ЭДОНакл"].

2.2. Если ничего не найдено - осуществляется переход к следующей строке. Если найдено - сохраняются значения Номер, Сумма и Дата.

2.3. Даты представлена в формате dd.mm.yyyy.

2.4. В столбцы из пункта 2.2 записываются найденные для данной строки значения.

2.5. В столбец "Сравнение дат" помещается "Не совпадает!", если найденная дата и дата накладной отличаются. Иначе - пустая строка.
Если по заданным условиям находится несколько подходящих документов - выбирается значение из первого найденного. Это означает, что если у документов разные суммы, берется та, что стоит первой в выгрузке из СБИС.
