**Расчет скользящего среднего количества решаемых задач на образовательной платформе**

 Расчет rolling_avg, который получается в результате усреднения столбца problems (количество решаемых на платформе в день задач) по 7 соседним измерениям. 
 
 При этом усреднение происходит не по последним 7, а по: предыдущим 3 элементам, текущему элементу, следующим 3 элементам.
 
_**Пример выполнения:**_

date  problems  rolling_avg

0    2022-01-01        83         0.00

1    2022-01-02        26         0.00

2    2022-01-03        55         0.00

3    2022-01-04       302       185.86

4    2022-01-05       503       218.43
