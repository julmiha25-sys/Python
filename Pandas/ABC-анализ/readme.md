**Функция для расчета ABC-анализа для данных с датафрейма pandas** (файл data.csv)

На вход подается датафрейм с любыми данными для анализа, в качестве значения для анализа используется индекс датафрейма (например, название препарата). В качестве расчетных значений - поля датафрейма (количество, розничная цена и т.п.) На выходе получается таблица с ABC-анализом.

**_Пример выполнения:_**

DR_NDrugs DR_Kol DR_CRoz ABC_DR_Kol ABC_DR_CRoz

ТЕРАФЛЮ ЛИМОН ОТ ГРИППА И ПРОСТУДЫ 22,1Г. №14 ПОР. Д/Р-РА Д/ПРИЕМА ВНУТРЬ ПАК. 7.0 15638.0 A A

НИМЕСИЛ 100МГ. 2Г. №30 ГРАН. Д/СУСП. Д/ПРИЕМА ВНУТРЬ ПАК. /ГУИДОТТИ/МЕНАРИНИ/ 2.8 14445.0 A A

ИНГАВИРИН 90МГ. №10 КАПС. /ВАЛЕНТА/ 18.0 13750.0 A A
