**XYZ-анализ продаж аптечной сети**

XYZ-анализ только для товаров, которые продавались более 1 дня за весь период.

Анализ производится на основе расчета коэф. вариации (ср.кв.отклонение/среднее арифм.)

Группа X - [0;10) 

Группа Y - [10;25)

Группа Z - [25;+∞)

На вход подается датафрейм с любыми данными для анализа, в качестве значения для анализа используется индекс датафрейма (например, название препарата). 

В качестве расчетных значений - поля датафрейма (количество, розничная цена и т.п.) На выходе получается таблица с XYZ-анализом.

**_Пример выполнения:_**

DR_NDrugs	                                         K_var	XYZ

0	L-ТИРОКСИН 50МКГ. №50 ТАБ. /БЕРЛИН ХЕМИ/        	0.00	X

1	L-ТИРОКСИН 75МКГ. №100 ТАБ. /БЕРЛИН ХЕМИ/       	0.47	Z

2	АЗАРГА 10МГ/МЛ.+5МГ/МЛ. 5МЛ. №1 ГЛ.КАПЛИ ФЛ./КАП.	0.00	X

3	АМОКСИЦИЛЛИН 500МГ. №16 КАПС. /ХЕМОФАРМ/	        0.47	Z

4	АРБИДОЛ МАКСИМУМ 200МГ. №10 КАПС. /ОТИСИФАРМ/Ф...	0.43	Z
