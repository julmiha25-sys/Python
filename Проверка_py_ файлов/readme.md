⭕**Проверка .py файлов (файлы user.py, etalon.py)** 

Имеются два файла .py - эталонный код и код пользователя. Данный кейс реализует систему, которая проверит их на заданных тестовых случаях и вернет соответствующие результаты.

**_Пример выполнения:_**

Тест не пройден

Ожидаемый результат: 3

Результат пользователя: -1

Тест не пройден

Ожидаемый результат: 7

Результат пользователя: -1

Тест не пройден

Ожидаемый результат: 11

Результат пользователя: -1

⭕**Проверка .py файла на вредоносный код (файлы user_malicious.py, etalon.py)** 

Добавление предварительной проверки кода пользователя на наличие вредоносного кода.

**_Пример выполнения:_**

Ошибка при проверке модуля user_malicious:

Код содержит вызов eval()
