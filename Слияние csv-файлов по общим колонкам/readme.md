⭕**Слияние csv-файлов по общим колонкам**

Скрипт для автоматической обработки и объединения файлов в единый выходной файл.

Скрипт реализован на базе функции process_files, которая принимает 2 значения: папку загрузки и папку для выгрузки. Результат выполнения функции – выходной файл combined_data.csv

Обрабатываемые файлы должны соответствовать шаблону: «ГГГГ-ММ-ДД-ЧЧ-ММ-idмагазина.csv»

В выходном файле строка заголовка - первая строка первого обработанного (валидного) файла.

В функции реализован анализ разделителя в загруженных CVS-файлах.

В выходном файле combined_data.csv в качестве разделителя используется запятая.

Функция анализирует колонки в исходных файлах и выбирает те, которые есть хотя бы в минимальном количестве файлов. Поиск валидных файлов осуществляется по правилу наличия хотя бы 50% общих колонок.

В выходном файле выполняется сортировка колонки с датой.
