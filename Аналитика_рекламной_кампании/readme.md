⭕2.Кейс2 (файл campaign_data.csv):

На основе csv-файла с данными о запущенных рекламных кампаниях необходимо определить:

1.Самую эффективную платформу по средним значениям конверсий

2.Самый успешный тип рекламной кампании по средним значениям конверсий

3.Самый успешный город проведения кампании по средним значениям конверсий

4.Доли затрат на рекламные кампании каждой платформы и каждого города

5.Средний бюджет всех кампаний, запущенных на определенной платформе

6.Генератор, генерирующий даты, в которые НЕ запускались кампании

7.Функция, которая принимает на вход путь к файлу и возвращает сгруппированные данные по городам, содержащие суммарный бюджет и количество кликов

8.Генератор, генерирующий данные о кампаниях, запущенных в указанном городе и имеющих бюджет выше заданного значения
